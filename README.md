# La Guita Grossa — guitagrossa.com

Web institucional de la comparsa **La Guita Grossa** de La Patum de Berga.

## Stack

- **Framework:** Astro 5 (SSG)
- **Estilos:** Tailwind CSS 4
- **Idioma:** Català
- **Hosting:** Pendiente (definirá el Board)

## Seccions

| Secció | Descripció |
|--------|-----------|
| Inici (Home) | Hero visual + slider de fotos, navegació principal |
| Història | Relat de la Guita Grossa amb galeria |
| Integrants | Tarjetes glass per categoria: Cap de Colla, Barristes, Guitaires, Col·laboradors |
| Contacte | Formulari d'email |

## Requisits

- Mobile-first, 100% responsive
- Lighthouse ≥90 performance
- Música ambiental amb control mute/play
- Paleta: vermell i verd (colors de La Guita)
- Footer: "Dissenyat amb ❤ per CIBAI Studio"

## Setup

```bash
npm install
npm run dev
```

## Estructura de branques

- `main` — producció (protegida, requereix PR aprovat)
- `develop` — integració
- `feature/*` — desenvolupament
